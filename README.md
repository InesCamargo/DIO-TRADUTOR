# 🧠 DIO – Tradutor com OpenAI

Este repositório contém a solução desenvolvida para o desafio da DIO, utilizando a API da OpenAI para criar um tradutor simples e funcional.

## 📂 Estrutura do Projeto

- **📸 Captura de tela**  
  Demonstra a implantação e funcionamento da função.

- **💻 Código-fonte (`tradutor.py`)**  
  Responsável por enviar o texto para tradução e retornar o resultado utilizando o modelo configurado.

- **📄 README**  
  Documento explicativo contendo detalhes do projeto, decisões e observações importantes.

## 🔒 Observações Importantes

As variáveis de ambiente **não foram publicadas**, pois contêm informações sensíveis (como a chave da API).  
Cada usuário deve configurar sua própria chave localmente.

## 🚀 Sobre o Processo

O desenvolvimento envolveu bastante aprendizado, ajustes e algumas aventuras entre **sinapses e bits**.  
A Copilot foi essencial para preencher lacunas de conhecimento e dar segurança durante a construção da solução.

Agradeço pela oportunidade de evoluir com este desafio!
